# 🔑 Password Generator

I made this password generator in html with the intention of practicing and grinding out the java script a little to start following it to the type script

## 🚀 Features

Custom Length: From 8 to 32 characters.

Security Options: Toggle uppercase, lowercase, numbers, and symbols.

Strength Indicator: Shows if your password is Weak, Medium, or Strong.

Copy to Clipboard: One-click button to copy the generated password.

Fully Responsive: Works fine on mobile and desktop.

## 🛠️ Technologies Used

HTML5 - Structure

CSS3 - Custom styling (Modern Card Design)

JavaScript - Password generation logic and DOM manipulation

# 🔑 Gerador de Senhas

Fiz esse gerador de senhas em HTML com a intenção de praticar e dar aquela "grindada" no JavaScript para daqui a pouco já começar a pular para o TypeScript.

## 🚀 Funcionalidades

Tamanho Customizado: De 8 a 32 caracteres.

Opções de Segurança: Filtros para maiúsculas, minúsculas, números e símbolos.

Indicador de Força: Mostra se a senha é Fraca, Média ou Forte.

Copiar para o Clipboard: Botão de um clique para copiar a senha gerada.

Totalmente Responsivo: Funciona de boa no celular e no PC.

## 🛠️ Tecnologias Usadas

HTML5 - Estrutura

CSS3 - Estilização (Design de Card Moderno)

JavaScript - Lógica de geração e manipulação do DOM
